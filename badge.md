# MPLS License Badges (v1.0)

## 1. Generic MPLS Badge

Use when you want to show the framework without a specific profile.

```md
![License: MPLS v1.0](https://img.shields.io/badge/license-MPLS_v1.0-blue)
```

---

## 2. MPLS Non-Commercial (NC)

```md
![License: MPLS-NC-1](https://img.shields.io/badge/license-MPLS--NC--1-red)
```

**Meaning:**
Free for personal & educational use. No commercial use.

---

## 3. MPLS Dual License

```md
![License: MPLS-DUAL-1](https://img.shields.io/badge/license-MPLS--DUAL--1-orange)
```

**Meaning:**
Free non-commercial, paid commercial use.

---

## 4. MPLS Donationware

```md
![License: MPLS-DON-1](https://img.shields.io/badge/license-MPLS--DON--1-yellow)
```

**Meaning:**
Free to use, donations encouraged.

---

## 5. MPLS OSS (MIT-style)

```md
![License: MPLS-OSS-MIT](https://img.shields.io/badge/license-MPLS--OSS--MIT-green)
```

**Meaning:**
Permissive open-source (MIT equivalent).

---

## 6. MPLS OSS (GPL-style)

```md
![License: MPLS-OSS-GPL](https://img.shields.io/badge/license-MPLS--OSS--GPL-brightgreen)
```

**Meaning:**
Copyleft open-source (GPL equivalent).

---

## 7. SPDX Explicit Badge (Advanced / Tooling)

Useful for tooling-heavy repos.

```md
![SPDX License](https://img.shields.io/badge/SPDX-LicenseRef--MPLS--DUAL--1-lightgrey)
```

---

## 8. Badge + Short Text (Recommended Pattern)

```md
![License: MPLS-DUAL-1](https://img.shields.io/badge/license-MPLS--DUAL--1-orange)

**License:** MPLS Dual License  
• Free for personal & educational use  
• Commercial use requires a paid license
```

---

## 9. Badge Decision Table (For Documentation)

| License      | Badge            |
| ------------ | ---------------- |
| MPLS-NC-1    | `MPLS--NC--1`    |
| MPLS-DUAL-1  | `MPLS--DUAL--1`  |
| MPLS-DON-1   | `MPLS--DON--1`   |
| MPLS-OSS-MIT | `MPLS--OSS--MIT` |
| MPLS-OSS-GPL | `MPLS--OSS--GPL` |

---

## 10. Optional: Custom Color Scheme (Consistent Meaning)

| Color  | Meaning          |
| ------ | ---------------- |
| Green  | Open Source      |
| Orange | Dual / Paid      |
| Red    | Non-Commercial   |
| Yellow | Donationware     |
| Blue   | Framework / Spec |
